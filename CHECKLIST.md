# Generation Checklist

The authoritative checklist lives in `references/checklist.md`.

Use it before claiming mobile store screenshot or featured graphic work is complete. It covers:

- source screenshot validation
- submission target, language, font, and color choices
- low source-count warnings
- straight/default layout rules
- App Store-only optional iPhone frame/notch/tilted choices
- Google Play featured graphic requirement
- exact pixel-dimension validation
